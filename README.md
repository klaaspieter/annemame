# annema.me

My personal website.

## Installation

Clone the repo:

```sh
git clone https://github.com/klaaspieter/annemame.git
```

## Develop

```sh
./bin/dev
```

## Deploy

Any push to `main` is build using `./bin/build` and deployed to GitHub pages
using [.github/workflows/main.yml](.github/workflows/main.yml).
